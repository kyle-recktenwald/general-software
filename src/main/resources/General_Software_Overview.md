# General Software Overview:

## Big O:

### Overview:
* Big O time is the language and metric we use to describe the efficiency of algorithms

  * Linked Lists
  * Trees, Tries, & Graphs

  * Queues
  * Heaps
  * Vectors
  * ArrayLists
  * Hash Tables
* **Algorithms:**
  * Breadth-First Search
  * Depth-First Search
  * Binary Search
  * Merge Sort
  * Quick Sort
* **Concepts:**
  * Bit Manipulation
  * Memory (Stack vs. Heap)
  * Recursion
  * Dynamic Programming
  * Big O (Time & Space)
